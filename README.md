# VICLOPS Technologies — Frontend Designs

Two React frontends for **VICLOPS Technologies Pvt Ltd**.

| Design | Purpose | Port | Theme |
|--------|---------|------|-------|
| **design-1** | Concept / alternate | 3001 | Dark (default) + **light theme toggle** |
| **design-2** | **Official company website** | 3002 | Dark blue & white, images from [viclops.com](https://www.viclops.com) |

## Quick Start

```bash
# Design 1 — with dark/light toggle (☀/☾ in navbar)
cd design-1
npm install
npm run dev

# Design 2 — official site
cd design-2
npm install
npm run dev
```

## Stack

React 18 · Vite 6 · Framer Motion 11
